디폴트 메서드(Default Method)에 대한 설명 중 올바른 것을 선택하시오

1) 디폴트 메서드를 선언하면 모든 클래스에서 해당 메서드를 반드시 재정의해야 함
2) 디폴트 메서드를 사용하면 인터페이스를 구현한 후 디폴트 메서드를 재정의하지 않은 모든 클래스에서 해당 메서드의 기본 구현이 사용됨
3) 디폴트 메서드는 인터페이스에 새로운 메서드를 추가할 때만 사용할 수 있으며, 기존 메서드의 시그니처를 수정하는 용도로 사용할 수 없음
4) 디폴트 메서드는 항상 코드 품질이 낮고 범용적이지 않으며, 사용을 권장하지 않음
5) 디폴트 메서드를 사용하여 새로운 인터페이스를 만들면 기존 구현체와 충돌하지 않으므로 편리하게 활용할 수 있음
